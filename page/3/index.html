Sublime-配置eslint

Posted on 2018-01-12   |   In FrontEnd

步骤

  • 全局安装eslint

    npm install -g eslint

完成后，在控制台输入eslint -v有版本号说明可以再npm中使用

  • 在sublime中通过packageControl安装 SublimeLinter 和 SublimeLinter-contrib-eslint两个包

  • 然后通过控制台使用eslint –init设置.eslintrc文件来规定你所需要的语法情况
