CSS_行级元素与块级元素小结

###行级元素与块级元素

块级元素：块状元素排斥其他元素与其位于同一行，可以设定元素的宽（width）和高（height），块级元素一般是其他元素的容器，可容纳块级元素和行内元素。
行级元素：行内元素不可以设置宽（width）和高（height），但可以与其他行内元素位于同一行，行内元素内一般不可以包含块级元素。行内元素的高度一般由元素内部的字体大小决定，宽度由内容的长度控制。

jQuery_列表展开折叠

###jQuery效果-toggle()方法
toggle方法可以切换元素的可见状态。
如果被选元素可见，则隐藏这些元素，若被选元素隐藏，则显示这些元素。

语法： 

$(selector).toggle(speed,callback,switch)

例子1： 

<script type="text/javascript">
      // 收缩展开效果
      $(document).ready(function(){  
      $("#content-obs").toggle(function(){ $("#content-obs").find("#arrow1").removeClass("arrow-right").addClass("arrow-down");  
      $("#obs").animate({height: 'toggle', opacity: 'toggle'}, "slow");},function(){ 
      $("#content-obs").find("#arrow1").removeClass("arrow-down").addClass("arrow-right");  
      $("#obs").animate({height: 'toggle', opacity: 'toggle'});
   });
</script>

Mobile_移动端解决页面点击穿透问题

移动端的click

最近遇到在移动端添加click事件进行进一步操作的问题。
常用的onclick在移动端表现并不够好，会有300ms延迟。
同时如果使用原生touch事件也存在点击穿透的问题，因为click是在touch系列事件发生后大约300ms才触发的，混用touch和click肯定会导致点透问题。
