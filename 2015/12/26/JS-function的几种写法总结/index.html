JS_function的几种写法总结

刚接触js方法的时候，感觉是十分混乱的，感觉上有很多种写法，今天总结几种function的相关写法。 具体参考博客http://www.cnblogs.com/see7di/archive/2011/06/18/2239716.html

最简单和基本的声明和定义

function func([parameters]){
    //content
}  // 直接函数声明function
var func = function([parameters]){
    //content
}; // 指派一个变量名给匿名函数的方式来定义它，定义函数表达式
  • 第二种写法，这个代码和前一个例子做了同样的事情。也许语法看起来比较奇怪，但它应该更能让你感觉到函数是一个对象，而且我们只是为这个对象指派了一个名称。

  • 当我们指派一个这样的函数的时候，我们并不一定要求必须是匿名函数。例如： 

    var func = function theFunc([parameters]) { 
    //content
} 
alert(func([parameters])); 
alert(theFunc([parameters])); // 结果与前面一样

  • 使用这种方式来定义函数在面向对象编程中是很有用的，可以使函数成为一个对象的属性 

    var myObject = new Object();
myObject.add = function(a,b) { return a+b }; 
// myObject 现在有一个叫做“add”的属性（或方法）
// 而且我能够象下面这样使用它
myObject.add(1, 2);

  • new 也可以定义一个函数

    var func = new Function([parameters], functionbody);

注：
1）new Function(…)使用了大写F，而不是小写f。 这就告诉javascript，我们将要创建一个类型是Function的对象
2）还要注意到，参数名和函数体都是作为字符串而被传递。我们可以随心所欲的增加参数，javascript知道函数体会是右括号前的最后一个字符串（如果没有参数，你能够只写函数体）

  • 函数：对象
    函数是javascript中的一种特殊形式的对象。他是一种类的数据类型，说明可以添加属性。
    当声明了函数，实际上js就是在后台创建了一个对象，对象的名为函数名，类型为function。
    同时还可以通过func.attribute的形式添加属性。

  • 指针
    因为function是一个对象，可以为一个function分配一个指针。如下例，变量ptr指向了对象myFunction。 

    function myFunction(message) { 
    alert(message);
}
var ptr = myFunction; // ptr指向了myFunction
ptr("hello"); // 这句会执行myFunction：输出"hello"

我们能够运行这个函数，就好像这个函数名已经被指针名代替了一样。所以在上面，这行ptr(“hello”); 和myFunction(“hello”);的意义是一样的。

  • 函数：原型
    每一个构造函数都有一个属性叫做原型(prototype)。这个属性可以为一个特定类声明通用的变量或者函数。
    原型在每一个构造函数中都有它的存在

    类的构造器

    function class(){
    //
} 
class.prototype = {}; 
var item = new class();

    闭包的作用

    (function(){ 
    //独立作用域 
})();

匿名函数

匿名函数就是没有名字的函数。
范例1：

(function(){

    alert("caibaojian.com");

})();

范例2：

var box = (function(){

    alert("caibaojian.com");

})();

范例3：

(function(n){

    alert(n);

})(100);
