JS-Undefined和Null的小结

Posted on 2016-04-15   |   In FrontEnd   |  

###判断undefined
JS中有两种特殊类型，undefined和null

var exp = undefined;
if (exp == undefined)
{
    alert("undefined");
}

以上面这种方法判断时，exp 为 null 时，也会得到与 undefined 相同的结果，虽然 null 和 undefined 不一样。

var exp = undefined;
if (typeof(exp) == "undefined")
{
    alert("undefined");
}

以上为正确判断方法。typeof 返回的是字符串，有六种可能：”number”、”string”、”boolean”、”object”、”function”、”undefined”

###判断null

var exp = null;
if (typeof(exp) == null)
{
    alert("is null");
}

exp 为 undefined 时，也会得到与 null 相同的结果，虽然 null 和 undefined 不一样。

var exp = null;
if (!exp)
{
    alert("is null");
}

如果 exp 为 undefined 或者数字零，也会得到与 null 相同的结果，虽然 null 和二者不一样。

var exp = null;
if (typeof(exp) == "null")
{
    alert("is null");
}

上面typeof总返回object，为了向下兼容。

所以正确判断方法为：

var exp = null;
if (!exp && typeof(exp)!="undefined" && exp!=0)
{
    alert("is null");
}
