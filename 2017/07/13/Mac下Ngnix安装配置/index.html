Mac下Ngnix安装配置

Posted on 2017-07-13   |   In FrontEnd

简介

Nginx(“engine x”)是一款是由俄罗斯的程序设计师所开发高性能的 Web和反向代理服务器，也是一个 IMAP/POP3/SMTP 代理服务器。
在高连接并发的情况下，Nginx是Apache服务器不错的替代品。

最近项目中遇到一些Nginx问题，正好有时间，想学一下，在自己的Mac环境下搭建一个Nginx环境试一试。

在Mac下，brew安装Nginx和PHP是非常方便的。
(https://brew.sh/)

Nginx安装与配置

安装Nginx

brew install nginx

修改配置文件

sudo vim /usr/local/etc/nginx/nginx.conf

修改端口号8080，为自己所用。

启动Nginx

sudo nginx #打开 nginx
nginx -s reload|reopen|stop|quit  #重新加载配置|重启|停止|退出 nginx
nginx -t   #测试配置是否有语法错误

PHP安装与配置

我安装的PHP版本为PHP5.4

安装php

brew install php54

安装完成以后，可以查看php配置文件，如下

The php.ini file can be found in:
   /usr/local/etc/php/5.4/php.ini
/usr/local/etc/php/5.4/php-fpm.conf

将php路径加入PATH

sudo vim ~/.bash_profile    
export PATH="$(brew --prefix php54)/bin:$PATH"

source ~/.bash_profile

配置Nginx支持php-fpm，修改项目根目录，为自己所用的项目路径

location / {
  root   /Users/liguanqun/Sites/workshop/nginxworkshop;
  index  index.html index.htm index.php;
  }


location ~ \.php$ {
      root          /Users/liguanqun/Sites/workshop/nginxworkshop;
      fastcgi_pass   127.0.0.1:9000;
      fastcgi_index  index.php;

      fastcgi_param  SCRIPT_FILENAME  $document_root$fastcgi_script_name;
      include        fastcgi_params;
      }

默认Nginx项目目录在/usr/local/Cellar/nginx/中，我们此处改成了自己的项目目录，运行index.php

<?php phpinfo();?>
